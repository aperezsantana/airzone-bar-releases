# AirzoneBar — releases

Builds de AirzoneBar (app de barra de menús para controlar un sistema Airzone por su API local).
El código fuente vive en un repo privado; aquí solo se publican los DMG.

Descarga: release **latest** → `AirzoneBar.dmg` (universal, macOS 13+).
Primera apertura: Ajustes del Sistema → Privacidad y seguridad → "Abrir de todas formas".
